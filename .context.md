# 🧠 AI SYSTEM INSTRUCTIONS FOR SIP-REGISTRAR-SERVICE
1. **Teknoloji:** Rust 1.84+ (Tokio, Tonic, Redis).
2. **Kalıcı Durum (State) Yasağı:** SIP kayıtları (Location Data) KESİNLİKLE PostgreSQL'e yazılamaz. Sadece Redis üzerinde TTL (Time-To-Live) ile geçici olarak tutulmalıdır.
3. **mTLS Strict:** gRPC bağlantılarında HTTP/Insecure kullanılamaz.
4. **Loglama & Trace:** Gelen `REGISTER` isteklerinde `x-trace-id` (SIP Call-ID) SUTS loglarına mutlaka eklenmelidir. `println!` kullanımı yasaktır.
